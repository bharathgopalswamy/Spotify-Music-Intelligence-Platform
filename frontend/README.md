# Spotify Music Intelligence Platform - React Version

This is the React/Vite version of the original HTML and CSS landing page.

## Run locally

```bash
npm install
npm run dev
```

## Build for production

```bash
npm run build
```

## Important asset paths

Place your actual files here so the existing links work:

- `public/screenshots/main-dashboard.png`
- `public/files/Spotify_Music_Intelligence_Dashboard.pdf`
- `public/dashboard/powerbi/Spotify_Music_Intelligence_Dashboard.pbix`
- `public/dashboard/powerbi/Spotify_Music_Intelligence_Dashboard.pbix.pdf`

Files inside `public` are served from the root path in Vite, so `public/screenshots/main-dashboard.png` becomes `/screenshots/main-dashboard.png`.
